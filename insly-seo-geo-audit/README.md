## Recommended execution order (highest ROI first)
1. **Technical fixes** (1 sprint): dedupe `/en/` URLs, noindex tag pages, add homepage H1, cross-link `ai.insly.com`. → `03-optimize/`
2. **Entity creation**: mint the Wikidata entity + `sameAs` + SoftwareApplication schema. → `05-cross-cutting/entity-optimization.md`, `02-build/schema-markup-generator.md`
3. **Meta + GEO quick wins**: rewrite the brand-only `Insly Pricing` title; add quotable blocks/FAQ. → `02-build/`
4. **Content gap fill**: mid-funnel commercial + comparison pages. → `01-research/content-gap-analysis.md`
5. **Instrument monitoring**: connect GSC + GA4 to convert N/A → Measured. → `04-monitor/`

## Biggest limitation
No Search Console / GA4 / paid SEO tool was connected this run. **Connecting Google Search Console (free) is the single largest unlock** — it converts rankings, impressions, CTR, indexing, and Core Web Vitals from N/A to Measured. Re-run the optimize + monitor phases afterward.
